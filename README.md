# FashionRepsSwiper

A **Tinder-style swiper** for browsing items from the [r/FashionReps trusted-sellers wiki](https://www.reddit.com/r/FashionReps/wiki/trusted/). Swipe right to save an item with its Yupoo link, left to skip.

Built for the community — fork it, improve it, run it locally. Pull requests welcome.

![Python 3.10+](https://img.shields.io/badge/python-3.10%2B-blue)
![License MIT](https://img.shields.io/badge/license-MIT-green)

## Features

- Swipeable card deck with drag, keyboard, and button controls
- **Multi-brand filter** — select one or many brands from the left sidebar
- **Saved items** with links back to the original Yupoo album
- Export saved list as **JSON** or **CSV**
- Resumable scraper (~1,450 store+brand searches + category fallback)
- Image proxy (Yupoo blocks hotlinking without a Referer header)

## Quick start (demo — no scrape)

Try the UI immediately with bundled sample data:

```bash
git clone https://github.com/WhemDev/FashionRepsSwiper.git
cd FashionRepsSwiper
pip install -r requirements.txt
python app/server.py --demo
```

Open **http://127.0.0.1:5000**

## Full setup (your own item database)

### 1. Install

```bash
pip install -r requirements.txt
```

### 2. Scrape items

The scraper reads `wiki_structure.json` (included) and searches each trusted Yupoo store for every brand it is listed under. It also checks **category folders** when search returns few results.

```bash
# Small test (~20 pairs, a few minutes)
python scraper/scrape.py --limit 20

# Full scrape (~1,450 pairs, 30–90 min, resumable)
python scraper/scrape.py
```

Progress is saved in `data/state.json` — you can stop and resume anytime.

To refresh the wiki seller list:

```bash
python scripts/fetch_wiki.py
```

### 3. Run the app

```bash
python app/server.py
```

Open **http://127.0.0.1:5000**. New items appear live while the scraper is still running.

### Scraper options

| Flag | Description |
|------|-------------|
| `--limit N` | Only process first N store+brand pairs |
| `--workers N` | Parallel workers (default 8) |
| `--rescan-brands "Gucci,Supreme"` | Force re-scrape specific brands |

## Controls

| Action | Input |
|--------|--------|
| Skip | Swipe left, **←**, or **X** button |
| Save | Swipe right, **→**, or heart button |
| Undo | **U** or **Ctrl+Z** |
| Filter brands | Click chips in the left sidebar (multi-select) |
| Collapse brands | Chevron next to "Brands" for a bigger card |

## Project layout

```
│   ├── server.py          # Flask API + image proxy
│   ├── mock_data/         # Sample data for --demo mode
│   └── static/            # Swipe UI (HTML/CSS/JS)

│   └── scrape.py          # Yupoo wiki scraper
├── scripts/

    ├── index.json
    └── img_cache/
```
